Aula de Git e GitHub

Comandos principais (principalmente após criar um novo projeto)

git init => toda vez que um projeto novo tiver sido criado e ainda não estiver utilizando o git

git add . => deve ser utilizado toda vez que uma alteração for feita em qualquer lugar do projeto (alterou código, apagou arquivos, criou ou inseriu arquivos novos)

git commit -m => salva todas as alterações feitas. Geralmente é acompanhado de uma mensagem descrevendo a alteração.

Exemplo: git commit -m"início do projeto"
